# ⚙️ My Cursor / VSCode Settings

> Personal IDE settings. Copy, paste, and code better.

---

## 🚀 How to Apply

**1. Open your IDE**
> VSCode or Cursor

**2. Open the Command Palette**
> `Ctrl + Shift + P`

**3. For `settings.json` — search for**
> `Preferences: Open User Settings (JSON)`

**For `keybindings.json` — search for**
> `Preferences: Open Keyboard Shortcuts (JSON)`

**4. Copy the file from this repository and paste into the corresponding file**

**5. Restart the IDE**

Done. ✅

---

## 📁 What's included

| File | Description |
|------|-------------|
| `settings.json` | Editor preferences, fonts, formatting, themes |
| `keybindings.json` | Custom keyboard shortcuts |

---

## 💡 Tips

- Always **backup your current settings** before replacing them
- Some settings may require extensions to be installed
- Restart the IDE after applying for all changes to take effect
